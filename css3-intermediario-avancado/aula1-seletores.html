Aula de Seletores Css

  • *{} - Seleciona todos os elementos da página
  • H1, h2, h3 {} - Seleciona os elementos h1,h2,h3...
  • .cor--verde{} - Seleciona qualquer elemento cujo atributo class tem o valor “verde”
  • p.verde{} - Seleciona somente elemento cujo atributo class tem o valor “verde”
  • #cabecalho{} - Seleciona o elemento cujo atributo id tem o valor “cabecalho”
  • li>a {} - Seleciona quaisquer elementos que são filhos de um elemento (mas não outros elementos a na página)
  • p a {} - Seleciona quaisquer elementos que residem dentro de um elemento, mesmo e houver outros elementos aninhados entre eles
  • h1+p {} - Seleciona o primeiro elemento depois de qualquer elemento (mas não outros elementos p)
  • h1~p {} - Se houvesse dois elementos que fossem irmãos de um elemento, essa regra se aplicaria ao dois
